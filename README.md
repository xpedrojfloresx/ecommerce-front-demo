# 🎮 Game Keys — E-commerce Front Demo

Proyecto integrador de front-end que simula una tienda online de **claves digitales de videojuegos para Steam**. Desarrollado por **Pedro J. Flores** como trabajo integrador N°1.

> 🌐 Demo en vivo → [xpedrojfloresx.github.io/ecommerce-front-demo](https://xpedrojfloresx.github.io/ecommerce-front-demo/)

---

## 🛠️ Stack Tecnológico

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura de páginas |
| CSS3 | Estilos y diseño responsive |
| JavaScript | Interactividad (carrito, modal, buscador) |
| GitHub Pages | Hosting del sitio estático |

> Proyecto 100% front-end estático, sin frameworks ni dependencias externas.

---

## 📁 Estructura del Proyecto

```
ecommerce-front-demo/

```

## 🚀 Cómo Ejecutarlo Localmente

No requiere instalación ni dependencias. Solo cloná el repo y abrí el archivo HTML:

```bash
# Clonar el repositorio
git clone https://github.com/xpedrojfloresx/ecommerce-front-demo.git

# Entrar al proyecto
cd ecommerce-front-demo

# Abrir en el navegador
open index.html
```

O simplemente arrastrá `index.html` a tu navegador.
